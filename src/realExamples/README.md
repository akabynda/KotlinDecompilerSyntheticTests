### Список существующих проектов

Для оценки комбинаций инструментов для декомпиляции и конвертации Java-кода в Kotlin был проведен поиск проектов на GitHub. При отборе учитывались репозитории с большим числом звезд и подходящими лицензиями. Выбраны следующие проекты:

- [**Coronavirus-Tracker**](https://github.com/sidhuparas/Coronavirus-Tracker)

- [**OpenNoteScanner**](https://github.com/allgood/OpenNoteScanner)

- [**ownCloud**](https://github.com/owncloud/android/tree/master)  

- [**SimpleStorage**](https://github.com/anggrayudi/SimpleStorage)  

- [**SpeedView**](https://github.com/anastr/SpeedView)
